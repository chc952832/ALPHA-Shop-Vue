# ALPHA SHOP Vue.js
使用Vue.js打造的商品結帳頁面

## 功能及特色
* 採用SPA方式，在切換頁面時有更好的使用者體驗
* 結帳表單區塊，根據步驟產生對應的樣式改變，並且運費可以反映到購物車區塊中
* 使用者寫入表單資料後，即使重整頁面，表單資料仍會留存
* 購物車區塊中可以藉由 + , - 按鈕來新增及減少購物車商品數量，並且自動計算總額
